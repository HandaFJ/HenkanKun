# 変換くん (一括画像最適化ツール)
ECサイト運営などにおける「画像アップロードの手間」と「ページ表示速度の改善」を解決するための、ブラウザ完結型最適化ツールです。

## 主な機能
- **複数画像の一括処理**: ドラッグ&ドロップで大量の画像を一度に最適化。
- **一括ダウンロード**: 処理後の画像をZip形式でまとめて保存。

## 使用技術
- **Frontend**: React (TypeScript)
- **Runtime**: Deno
- **Build Tool**: Vite
- **Libraries**: 
  - `JSZip`: 画像のパッケージング
  - `Canvas API`: 高速な画像リサイズ処理
